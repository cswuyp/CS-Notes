# 虚函数的实现原理

我们可以从虚函数表和虚函数指针来了解虚函数的实现原理。
简单地说，每一个含有虚函数（无论是其本身的，还是继承而来的）的类都至少有一个与之对应的虚函数表，其中存放着该类所有的虚函数对应的函数指针。
